## The handanim Library

handanim is a Python animation library that creates whiteboard-style educational videos using Cairo graphics. It simulates hand-drawn sketches with configurable roughness and fills.

### Complete API Reference

#### Core Imports Structure
```python
from handanim.core import (
    Scene,                    # Main animation container
    SketchStyle,             # Style for roughness/bowing
    StrokeStyle,             # Line color, width, opacity
    FillStyle,               # Fill color, pattern, hatching
    DrawableGroup,           # Group multiple drawables
    CompositeAnimationEvent,  # Chain multiple animations
)
from handanim.animations import (
    SketchAnimation,         # Progressive drawing effect
    FadeInAnimation,         # Fade from transparent
    FadeOutAnimation,        # Fade to transparent
    ZoomInAnimation,         # Scale from 0
    ZoomOutAnimation,        # Scale to 0
    TranslateToAnimation,    # Move to coordinates
    TranslateFromAnimation,  # Move from coordinates
    TranslateToPersistAnimation,  # Move and stay
)
from handanim.primitives import (
    Line, Arrow, Curve, CurvedArrow,
    Polygon, Rectangle, Square, RoundedRectangle, RoundedSquare,
    Ellipse, Circle, NGon,
    Text, Math, Eraser, SVG, VectorSVG,
    Image, Video
)
from handanim.stylings.color import (
    BLACK, WHITE, RED, BLUE, GREEN, ORANGE, YELLOW,
    GRAY, LIGHT_GRAY, SKY_BLUE, ERASER_HINT_COLOR
)
```

### Animation Events - Complete

#### Timeline Management
All animations use ABSOLUTE time from video start (seconds).
- `start_time`: When animation begins
- `duration`: How long the animation takes
- Animations persist on screen after completion unless hidden/faded

#### SketchAnimation (Most Common)
```python
# Draw the object progressively, then fill it
scene.add(
    SketchAnimation(start_time=6, duration=3),
    drawable=triangle
)

# With fill delay
scene.add(
    SketchAnimation(
        start_time=6, 
        duration=3,
        data={"wait_before_fill": 0.5}  # Seconds between stroke and fill
    ),
    drawable=triangle
)
```

#### Fade Animations
```python
scene.add(FadeInAnimation(start_time=2, duration=1), drawable=label)
scene.add(FadeOutAnimation(start_time=5, duration=1.5), drawable=old_content)
```

#### Zoom Animations
```python
scene.add(ZoomInAnimation(start_time=3, duration=2), drawable=small_circle)

```

#### Translate Animations
```python
# Move TO a specific position
target_pos = (1200, 500)
scene.add(
    TranslateToAnimation(
        start_time=5, 
        duration=2,
        data={"point": target_pos}
    ),
    drawable=moving_circle
)

# Move FROM a position (object starts there, moves to current position)
scene.add(
    TranslateFromAnimation(
        start_time=5,
        duration=2,
        data={"point": start_pos}
    ),
    drawable=circle
)

# Translate and persist at new position (permanent move)
scene.add(
    TranslateToPersistAnimation(
        start_time=5,
        duration=2,
        data={"point": new_position}
    ),
    drawable=object
)
```

### Grouping and Composition

#### DrawableGroup
Group multiple related drawables to animate together:
```python
labels = [("a", (450, 600)), ("b", (700, 800))]
label_group = DrawableGroup(
    elements=[
        Math(tex_expression=f"${label}$", position=pos, font_size=96)
        for label, pos in labels
    ]
)
scene.add(SketchAnimation(start_time=8, duration=2), drawable=label_group)
```

#### CompositeAnimationEvent
Chain or parallelize animations on the same drawable:
```python
scene.add(
    CompositeAnimationEvent(
        events=[
            # Draw it first
            SketchAnimation(start_time=0, duration=2),
            # Then fade it out after 2 more seconds
            FadeOutAnimation(start_time=4, duration=1),
        ]
    ),
    drawable=temporary_label
)

# Complex: Draw -> Translate while zooming out
scene.add(
```

### Transformations (Applied to Drawables)

Transformations return NEW drawables (immutable):
```python
# Translate (offset from current position)
moved_triangle = triangle.translate(offset_x=100, offset_y=50)

# Scale (from origin or relative)
big_triangle = triangle.scale(scale_x=1.5, scale_y=1.5)

# Rotate (angle in degrees)
rotated_arrow = arrow.rotate(angle=45)

# Chaining transformations
final = triangle.translate(100, 50).rotate(30).scale(1.2, 1.2)
```

## Code Generation Guidelines

1. **Use relative imports**: All examples use from `handanim.X import ...`
2. **FONT_NAME = "feasibly"**: Use this as default font
3. **Coordinates**: Use world coordinates (viewport handles scaling to screen)
4. **Timing**: Space animations with ~0.5s gaps between major scenes
5. **Groups**: Use DrawableGroup for related elements (like labels)
6. **Eraser**: Use Eraser with SketchAnimation to clear between major topics
7. **Math expressions**: Always use raw strings `r"$...$"` for LaTeX
8. **Scene dimensions**: Standard is `Scene(width=1920, height=1088)`
